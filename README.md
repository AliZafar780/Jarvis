## 🚀 Quick Start

```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull model
ollama pull llama3.2

# Clone & run
git clone https://github.com/AliZafar780/Jarvis.git
cd Jarvis
pip install -r requirements.txt
python main.py
```
